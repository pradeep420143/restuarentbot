# Alpha Restaurant AI Ordering System 🍽️

A smart restaurant ordering system built with Streamlit that understands natural language orders.

## Features
- Natural language processing for orders
- Veg and Non-Veg menu options
- GST calculation (5%)
- Table number tracking via QR codes
- User-friendly interface

## How to Use
1. Scan QR code with table number
2. Select menu type
3. Enter order naturally (e.g., "2 chicken biryani and 1 fish curry")
4. Click "Place Order"
5. View order summary with GST
